
# Employee-Management-System 👨‍💼

A simple Employee Management System developed using Java, Servlets, JDBC, HTML, CSS, and MySQL.

This project is created to learn Java Web Development concepts and currently includes Login and Registration functionalities.

---

# 🚀 Features

- Employee Registration
- Employee Login
- JDBC Database Connectivity
- Servlet-Based Backend
- Form Validation
- Authentication System
- Simple and Clean UI

# ▶️ How to Run the Project

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/Employee-Management-System.git
````

## 2️⃣ Open Project

Open the project in:

* VS Code
* Eclipse IDE

## 3️⃣ Configure Apache Tomcat

* Install Apache Tomcat
* Add Tomcat Server to IDE
* Configure Server Runtime

## 4️⃣ Configure MySQL Database

### Create Database

```sql
CREATE DATABASE employee_db;
```

### Use Database

```sql
USE employee_db;
```

### Create Employee Table

```sql
CREATE TABLE employee (
    id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100),
    email VARCHAR(100),
    password VARCHAR(100)
);
```

## 5️⃣ Run the Project

Start Tomcat Server and open:

```bash
http://localhost:8080/EmployeeDemo/Login.html
```
